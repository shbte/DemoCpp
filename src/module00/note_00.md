### Const

#### C语言中 : 00

```
1、const使得变量具有只读属性(array[const int]：不可行)

2、const将具有全局生命周期的变量存储于只读存储区

3、const不能定义真正意义上的常量

4、c语言真正意义上的常量只有枚举
```

#### C++语言中 : 00 ~ 01

```
1、当碰见const声明时，将常量放入符号表(array[const int]：可行)

2、后续使用常量时，则直接在符号表中的寻找值(符号表中的值不变，但是空间的值却变了)

3、对const常量使用&( || 使用extern修饰在其它文件使用)时，才会分配空间

4、const修饰的是真正意义的常量，存放在编译器的符号表中。

    const常量去符号表中取值，指针去常量空间找值。

5、const常量和define(宏定义)不一样

    const常量由编译器处理，进行类型检查和作用域检查

    define由预处理器处理，单纯的文本替换

6、const把变量和常量值变为只读变量，但是可以使用指针法还改变const的值

    const不止可以修饰变量，也可以修饰常量值
```
